1. **Implement procedures** under `packages/api/src/routers/admin/`:
   - **categories.ts**:
     - `list`: returns all (incl. inactive) ordered by displayOrder.

     - `reorder`: { orderedIds: string[] } — single transaction updating displayOrder = index+1 for each id.
     - `toggleActive`: { id }.

   - **geo/countries.ts**:
     - `list`, `create`, `update`, `reorder`, `toggleActive`, `delete` (refuses if cities exist; admin must delete cities first).
     - Fields: code (ISO 3166-1 alpha-2, unique, uppercase), nameRu/En/Uk, flagEmoji (single emoji), displayOrder, isActive.
   - **geo/cities.ts**:
     - `list`: { countryId?, search?, cursor?, limit }.
     - `create`: { countryId, nameRu/En/Uk, displayOrder? }.
     - `update`, `delete` (refuses if Partner.cityId references it; admin must reassign).
     - `reorder`: { countryId, orderedIds[] }.
