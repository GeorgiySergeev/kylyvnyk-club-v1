1. **Implement procedures** in `packages/api/src/routers/admin/chat.ts`:
   - `listConversations`: { cursor?, limit default 30, max 100, status?: 'all'|'OPEN'|'PENDING'|'CLOSED', assignedToId?: 'me'|'unassigned'|adminUserId, search? (matches latest message body or member name/email), sort?: 'newest'|'oldestUnread' }.

   - `listMessages`: { conversationId, cursor?, limit } — same shape as member side.

   - `bulkClose`: { conversationIds[] } — close many at once.
   - `getMyPresence`: returns active operator presence info (see step 3).
